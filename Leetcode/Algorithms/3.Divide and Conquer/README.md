# 分治法(divide and conquer) #
***
### 分治思想 ###
	对每一层进行循环递归，每一层有三个步骤：
	1、分解：将原问题分解为若干个规模较小，相对独立，与原问题形式相同的子问题
	2、解决：若子问题规模较小且易于解决时，直接求解，否则递归解决子问题
	3、合并：将子问题的解和并为原问题的解

### 典型例题 ###
>+ 二分查找
>+ 归并排序 
>+ 快速排序
>+ 数列中的最长连续序列的最大和序列
